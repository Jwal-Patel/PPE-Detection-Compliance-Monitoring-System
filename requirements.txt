# =============================================================================
# TROUBLESHOOTING
# =============================================================================
#
# Issue: "ModuleNotFoundError: No module named 'torch'"
# Solution: pip install torch torchvision
#
# Issue: "CUDA out of memory" errors
# Solution: Use CPU mode or reduce batch size in train.py
#
# Issue: OpenCV on headless servers
# Solution: pip install opencv-python-headless (instead of opencv-python)
#
# Issue: PDF generation fails (reportlab)
# Solution: Ensure reportlab is installed: pip install reportlab
#
# Issue: Permission denied on .env file
# Solution: chmod 600 .env (Linux/Mac) or update file permissions
#
